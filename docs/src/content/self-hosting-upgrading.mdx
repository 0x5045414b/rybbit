# Upgrading

To upgrade your self-hosted Rybbit instance to the latest version:

<Steps>
### 1. Pull the latest changes from the repository:
```bash
git pull
```

### 2. Rebuild and restart your Docker containers:
```bash
docker compose up -d --build
```
</Steps>

We add great features to Rybbit all the time, so upgrade often to get your hands on the latest and greatest!
